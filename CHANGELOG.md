# Changelog

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-04-30

### Added
- `MkToastyConfig` ScriptableObject with all configurable parameters
- `MkToastyPresenter` with slide-in/out animation and audio playback
- `MkToastyTriggerController` with keyboard shortcut, random timer, and code triggers
- Support for all four screen corners
- `OnShown` / `OnHidden` events on `MkToastyPresenter`
- New Input System support (Unity Input System 1.7+)
